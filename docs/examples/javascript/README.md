# JavaScript Examples

## Setup

```bash
npm install
```

## Configure endpoint (optional)

```bash
export AIGEN_RPC_URL=http://localhost:9944
export AIGEN_WS_URL=ws://localhost:9944
```

## Run

Basic JSON-RPC calls:

```bash
npm run basic
```

Transaction submission example:

```bash
npm run tx
```

WebSocket subscription example:

```bash
npm run ws
```
